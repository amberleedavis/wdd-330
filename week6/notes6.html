A form is basically something you fill out on a website. This can include filling out any login information, search boxes, comment boxes, phone numbers, or even drop down menus. Forms are created in HTML and then linked to a JavaScript file to give each form a purpose. To create forms, you use input fields, buttons, and select menus in the HTML.
